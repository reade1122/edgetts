## 系統簡介

語音報讀系統個人版是一個專為學習障礙學生設計的 PDF 語音報讀工具，支援多種朗讀模式，並提供 PDF 標記功能，讓學生可以點選預先標記的範圍直接發音。

### 主要特色
- ✅ 支援本地 PDF 檔案和 Google 雲端硬碟檔案
- ✅ 多種朗讀模式：整頁朗讀、框選朗讀、觸控選取朗讀
- ✅ PDF 標記功能：手動標記、AI 自動標記
- ✅ 多種語音選擇：本機語音、Edge TTS 語音
- ✅ 可調整語速和音量
- ✅ 支援圖片辨識（需設定 API KEY）

## 快速開始

### 1. 載入 PDF 檔案

#### 方法一：載入本地檔案
1. 點選左上角的「選擇檔案」按鈕
2. 從電腦中選擇要載入的 PDF 檔案
3. 系統會自動載入並顯示第一頁

#### 方法二：載入 Google 雲端硬碟檔案
1. 在網址列輸入 Google 雲端硬碟檔案的分享連結
2. 系統會自動解析並載入檔案

**注意事項：**
- 雲端檔案必須設定為「知道連結的使用者」可檢視
- 標記功能僅限本地 PDF 檔案使用

### 2. 開始朗讀

載入 PDF 後，您可以使用以下方式進行朗讀：

- **整頁朗讀**：點選右下角的「整頁朗讀」按鈕
- **框選朗讀**：使用可移動的框選區域選擇要朗讀的內容
- **觸控選取**：勾選「觸控選取」後，用手指或滑鼠選取文字進行朗讀

## 主要功能

### 一、朗讀功能

#### 1. 整頁朗讀
- **位置**：右下角「整頁朗讀」按鈕
- **功能**：自動朗讀當前頁面的所有文字內容
- **操作**：
  1. 點選「整頁朗讀」按鈕
  2. 按鈕會變成「取消朗讀」，可隨時點選停止
  3. 系統會依序朗讀頁面中的每個句子

#### 2. 框選朗讀
- **位置**：頁面上的可移動框選區域
- **功能**：朗讀框選區域內的文字和圖片內容
- **操作**：
  1. 調整框選區域的位置和大小
  2. 點選框選區域內的「朗讀」按鈕
  3. 系統會朗讀框選區域內的所有內容（包含圖片辨識）

#### 3. 觸控選取朗讀
- **位置**：左側「觸控選取」勾選框
- **功能**：手動選取文字進行朗讀
- **操作**：
  1. 勾選「觸控選取」
  2. 使用手指或滑鼠拖曳選取要朗讀的文字
  3. 選取的文字會以紅色標示
  4. 點選右下角「朗讀選取文字」按鈕開始朗讀
  5. 點選「清除選取文字」可清除選取

### 二、PDF 標記功能

標記功能讓您可以預先在 PDF 上標記特定區域，之後點選標記區域即可直接發音。

#### 進入標記模式
1. 點選左上角「⚙️」設定按鈕
2. 在「三、PDF 標記功能」區塊中點選「標記區塊」
3. 系統會進入標記模式，游標會變成十字準星

**重要提醒：**
- 標記功能僅限本地 PDF 檔案使用
- 雲端檔案無法進行標記編輯
- 如需使用標記功能，請先下載 PDF 到本地，然後重新上傳

#### 手動標記
1. 進入標記模式後，點選工具欄的「手動標記」按鈕
2. 在 PDF 上拖曳滑鼠或手指，框選要標記的區域
3. 標記框會以藍色邊框顯示
4. 可以調整標記框的大小和位置

#### AI 標記（需設定 API KEY）
1. 進入標記模式後，點選工具欄的「AI標記」按鈕
2. 選擇「新增標記」或「重新標記」
3. 系統會自動分析 PDF 內容並標記可朗讀的區域
4. AI 標記會自動識別文字和圖片區域

#### 標記框操作
每個標記框都有以下功能按鈕（懸停在標記框上時顯示）：

- **🔒 鎖定/解鎖**：鎖定後無法移動、調整或刪除標記框
- **✏️ 編輯替換文字**：設定該標記區域的替換文字
- **🗑️ 刪除**：刪除該標記框

#### 保存和匯出標記
1. 標記完成後，點選「保存標記」將標記資料儲存到 PDF 檔案中
2. 點選「匯出PDF」下載包含標記的 PDF 檔案
3. 下次載入該 PDF 時，標記會自動恢復

#### 使用已標記的 PDF
1. 載入已標記的 PDF 檔案
2. 直接點選標記區域即可發音
3. 如果標記有設定替換文字，會使用替換文字進行朗讀

## 詳細操作說明

### 頁面操作

#### 翻頁
- **上一頁**：點選頂部的「上一頁」按鈕
- **下一頁**：點選頂部的「下一頁」按鈕
- **頁碼顯示**：頂部中央會顯示當前頁碼

### 語音設定

#### 選擇語音
- **位置**：右上角語音選擇下拉選單
- **選項**：
  - 本機語音（中文女聲）
  - 本機語音（英文女聲）
  - Edge TTS 語音（多種語音可選）

#### 調整語速
- **位置**：語音選擇下方的語速滑桿
- **範圍**：-50% 到 +50%
- **操作**：拖曳滑桿調整語速

#### 調整音量
- **位置**：語速滑桿下方的音量滑桿
- **範圍**：0% 到 100%
- **操作**：拖曳滑桿調整音量

### 標記框編輯

#### 移動標記框
1. 將滑鼠或手指放在標記框上
2. 按住並拖曳即可移動位置

#### 調整標記框大小
1. 將滑鼠移到標記框的四個角落或邊緣
2. 游標會變成調整大小圖示
3. 拖曳即可調整大小

#### 編輯替換文字
1. 懸停在標記框上，點選「✏️ 編輯」按鈕
2. 在彈出的視窗中：
   - 「原始文字」欄位：顯示標記區域的原始文字（點選可試聽）
   - 「替換文字」欄位：輸入要替換的文字（旁邊有「預覽」按鈕可試聽）
3. 點選「確定」儲存，或「取消」放棄修改

#### 鎖定標記框
1. 懸停在標記框上，點選「🔒 鎖定」按鈕
2. 鎖定後，標記框無法移動、調整或刪除
3. 再次點選可解鎖

## 進階設定

### API KEY 設定

API KEY 用於以下功能：
- AI 圖片辨識（框選朗讀時識別圖片內容）
- AI 標記功能（自動標記 PDF 可朗讀區域）

#### 取得 API KEY
1. 點選左上角「⚙️」設定按鈕
2. 在「一、API KEY 設定」區塊中點選「？」按鈕查看教學
3. 連線到 [Google AI Studio](https://aistudio.google.com/u/2/api-keys)
4. 登入 Google 帳號
5. 依照教學圖片建立並複製 API KEY

#### 設定 API KEY
1. 在「API KEY」輸入框中貼上您的 API KEY
2. 或使用「📷 掃描 QR Code」功能掃描 QR Code
3. 點選「儲存」按鈕
4. 系統會將 API KEY 儲存在瀏覽器中

#### 刪除 API KEY
- 點選「刪除 API KEY」按鈕即可清除已儲存的 API KEY

### PDF 快取管理

系統會自動將載入的 PDF 檔案快取到瀏覽器中，以加快下次載入速度。

#### 清除快取
1. 點選左上角「⚙️」設定按鈕
2. 在「二、PDF 快取」區塊中點選「清除 PDF 快取」
3. 系統會清除所有快取的 PDF 檔案

### 標記密碼保護

為了保護標記內容，系統支援設定密碼保護。

#### 設定密碼
1. 進入標記模式
2. 點選工具欄的「設定密碼」按鈕
3. 輸入密碼並確認
4. 之後進入標記模式時需要輸入密碼

#### 取消密碼保護
1. 進入標記模式（需輸入密碼）
2. 點選「設定密碼」按鈕
3. 在密碼輸入框中留空並確認
4. 即可取消密碼保護

## 常見問題

### Q1: 為什麼無法標記雲端 PDF 檔案？
**A:** 標記功能僅限本地 PDF 檔案使用。如需使用標記功能，請：
1. 下載雲端 PDF 檔案到本地
2. 使用「選擇檔案」功能上傳本地檔案
3. 即可進行標記操作

### Q2: 標記後如何保存？
**A:** 標記完成後，請依序執行：
1. 點選「保存標記」將標記資料儲存到 PDF
2. 點選「匯出PDF」下載包含標記的 PDF 檔案
3. 下次載入該 PDF 時，標記會自動恢復

### Q3: 為什麼 AI 標記功能無法使用？
**A:** AI 標記功能需要設定 Gemini API KEY。請：
1. 點選「⚙️」設定按鈕
2. 在「一、API KEY 設定」中設定您的 API KEY
3. 設定完成後即可使用 AI 標記功能

### Q4: 標記框無法移動或調整？
**A:** 請檢查標記框是否已鎖定：
1. 懸停在標記框上查看是否有鎖定圖示
2. 如果已鎖定，點選鎖定按鈕解鎖
3. 解鎖後即可移動和調整

### Q5: 如何清除所有標記？
**A:** 
1. 進入標記模式
2. 點選「清除標記」按鈕
3. 確認後會清除當前頁面的所有標記

### Q6: 標記框的工具欄按鈕看不到？
**A:** 
- 將滑鼠懸停在標記框上，工具欄會自動顯示
- 如果標記框太小，工具欄會顯示在框的上方

### Q7: 如何試聽替換文字？
**A:** 
1. 點選標記框的「✏️ 編輯」按鈕
2. 在「替換文字」欄位旁點選「預覽」按鈕
3. 系統會使用首頁設定的語音和語速進行試聽

### Q8: 支援哪些檔案格式？
**A:** 目前僅支援 PDF 格式。不支援圖片格式的 PDF（掃描檔），但可透過 AI 圖片辨識功能讀取圖片中的文字。
